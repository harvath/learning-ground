Quick "is this dated" check

If you hear or read any of these presented as current, treat it as a signal the source is stale:

  • AutoGPT / BabyAGI as the state of the art in autonomous agents — this was the 2023 hype wave, superseded.
  • "Prompt engineer" as a distinct, standalone job title — the skill remains in demand but got absorbed into broader AI/agent engineering roles.
  • OpenAI Swarm as OpenAI's agent framework — archived, superseded by the OpenAI Agents SDK (Mar 2025).
  • Microsoft AutoGen as a standalone active framework — folded into Microsoft Agent Framework (GA Apr 2026).
  • "Million-token context windows have made RAG obsolete" — the 2024-era claim; 2026 empirical work shows RAG still wins on cost and accuracy at scale for frequently-changing knowledge.
