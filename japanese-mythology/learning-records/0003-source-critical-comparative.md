# 史料批判・比較神話・世界構造の視点で問う段階に到達

第4セッションで、ユーザーは自発的に3種の高次の問いを出した：①古事記と日本書紀で扱いが違うのでは／どちらに錨を下ろすのが正解か（史料批判）、②イザナキ黄泉行きのオルフェウス酷似は伝播か人間の普遍傾向か（比較神話学）、③黄泉は天／地上／地下のどれで、島根の黄泉比良坂と辻褄は合うのか（神話世界の空間構造）。

筋の理解を超えて、テクストの成り立ち・他文化との関係・宇宙論の整合性を問うている＝かなり高い到達点。

**含意**：
- 今後、神話素には「どの史料のどのバージョンか」を添える。既定は古事記アンカー＋日本書紀の一書を異伝として差し込む（[[MISSION.md]] の"作品を吟味する目"に直結）。→ [[kiki-no-chigai]] reference 新設。
- 黄泉の空間（高天原／葦原中国／黄泉／根の国／常世）は綺麗な三層でない点を、出雲編（L0005前後）で正式に図解予定。出雲＝異界＝国津神領域という含みが国譲りの伏線になる。
